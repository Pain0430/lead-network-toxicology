# 🔬 铅诱导高血压机制及干预策略

## 研究背景

基于NHANES数据分析发现：**收缩压是铅诱导CKM综合征的关键中介变量 (中介效应88.6%)**

## 核心发现

### 铅 → 血压 → CKM 机制

```
铅暴露
   ↓
┌────────────────────────────────────────────┐
│  通路1: RAS系统                            │
│  铅 → ACE激活 → Ang II → 血管收缩 → 血压↑ │
└────────────────────────────────────────────┘
   ↓
┌────────────────────────────────────────────┐
│  通路2: 氧化应激                           │
│  铅 → ROS增加 → NOS失活 → NO↓ → 血压↑    │
└────────────────────────────────────────────┘
   ↓
┌────────────────────────────────────────────┐
│  通路3: 炎症反应                           │
│  铅 → NF-κB激活 → IL-1β/TNF → 血压↑     │
└────────────────────────────────────────────┘
```

## 🎯 关键靶点分析

### 1. ACE (血管紧张素转换酶)

| 属性 | 说明 |
|------|------|
| 类型 | 金属蛋白酶 |
| 金属位点 | **Zn2+结合位点** |
| 现有药物 | 卡托普利、赖诺普利 (ACE抑制剂) |
| 干预思路 | 金属螯合剂竞争性抑制 |

### 2. NOS3 (内皮型一氧化氮合酶)

| 属性 | 说明 |
|------|------|
| 类型 | 氧化还原酶 |
| 金属位点 | **Zn2+, Fe2+ (heme)** |
| 辅因子 | BH4 (四氢生物蝶呤) |
| 干预思路 | 补充L-精氨酸、BH4 |

### 3. SOD1/CAT (抗氧化酶)

| 属性 | 说明 |
|------|------|
| 类型 | 氧化还原酶 |
| 金属位点 | **Cu+/Zn2+ (SOD), Fe (CAT)** |
| 干预思路 | 金属替代疗法、抗氧化剂 |

### 4. 炎症通路

| 靶点 | 类型 | 现有抑制剂 |
|------|------|-----------|
| IL1B | 细胞因子 | 阿那白滞素 |
| TNF | 细胞因子 | 依那西普 |
| NFKB1 | 转录因子 | 姜黄素、白藜芦醇 |

## 💊 干预策略

### 策略一：老药新用

| 药物类别 | 代表药物 | 机制 |
|---------|---------|------|
| ACE抑制剂 | 卡托普利 | 抑制ACE，降低Ang II |
| ARB类 | 氯沙坦 | 阻断AT1受体 |
| CCB类 | 氨氯地平 | 钙通道阻滞 |

### 策略二：抗氧化

| 化合物 | 靶点 | 证据 |
|--------|------|------|
| N-乙酰半胱氨酸 | 谷胱甘肽前体 | 研究中 |
| 辅酶Q10 | 线粒体抗氧化 | 临床试验 |
| 维生素C/E | 抗氧化 | 辅助治疗 |

### 策略三：抗炎

| 化合物 | 靶点 | 证据 |
|--------|------|------|
| 姜黄素 | NF-κB | 临床前 |
| 白藜芦醇 | SIRT1/NF-κB | 临床前 |
| 黄酮类 | 炎症 | 天然产物 |

## 🔬 VCell模拟建议

### 模型1: RAS通路模型

**组件:**
- Species: Lead, ACE, AngI, AngII, AGTR1, Ca2+, VascularTone
- Reactions: 
  - Lead → ACE (激活)
  - ACE + AngI → AngII
  - AngII + AGTR1 → VascularTone

**参数来源:** 文献中ACE动力学参数

### 模型2: 氧化应激-NO通路

**组件:**
- Species: Lead, ROS, NOS3, BH4, NO, VascularTone
- Reactions:
  - Lead → ROS (产生)
  - ROS + NOS3 → NOS3_inactive
  - NOS3 + BH4 → NO

### 模型3: 炎症通路

**组件:**
- Species: Lead, NFKB, IL1B, TNF, EndothelialFunction
